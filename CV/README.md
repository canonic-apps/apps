# CV

Governed academic CV generator.

inherits: /canonic-apps/CV/

---

## Structure

```
cv.md
├── Identity          <- Name, credentials, links
├── Education         <- Degrees (Evidence: transcripts)
├── Positions         <- Jobs (Evidence: offer letters)
├── Publications      <- Papers (Evidence: DOI, PMID)
├── Co-Authors        <- Collaborators (Evidence: refs.bib)
├── Grants            <- Funding (Evidence: award letters)
├── IP                <- Patents, IDFs
├── Mentors           <- Advisors
├── Honors            <- Awards
├── Talks             <- Presentations
├── Organizations     <- Affiliations
└── Timeline          <- Chronological summary
```

---

## Artifacts

```
{name}-cv.md  → source (governed, living)
{name}-cv.pdf → output (styled, exportable)
```
