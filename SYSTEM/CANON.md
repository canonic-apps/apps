## Axioms

## Constraints

- SYSTEM apps MUST be universal.
- SYSTEM apps MUST NOT require opt-in.
- SELF MUST be the base (all others inherit).
- Every user MUST have SYSTEM apps.
- AGENT registry MUST be R-scoped.
