## Purpose

## Categories

| Category | Assessment | Cancer Risk |
|----------|------------|-------------|
| 0 | Incomplete | Needs additional imaging |
| 1 | Negative | ~0% |
| 2 | Benign | ~0% |
| 3 | Probably Benign | <2% |
| 4 | Suspicious | 2-95% |
| 5 | Highly Suggestive | >95% |
| 6 | Known Biopsy-Proven | 100% |
