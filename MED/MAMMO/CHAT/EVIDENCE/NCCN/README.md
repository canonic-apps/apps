## Key References

### DCIS vs Invasive (Minh's concern)

```
DCIS (Ductal Carcinoma In Situ):
├── Non-invasive
├── Treatment: Surgery +/- radiation
├── Endocrine therapy optional
└── NO systemic chemotherapy

Invasive Breast Cancer:
├── May require systemic therapy
├── Staging determines treatment
├── Chemo based on molecular subtype
└── Different prognosis
```

**VALIDATOR MUST differentiate these pathways.**
