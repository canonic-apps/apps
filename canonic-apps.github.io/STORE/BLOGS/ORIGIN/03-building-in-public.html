The Easy Path

We could have kept it closed.

Seven years of development. Millions in funding. Clinical trials. Production deployments. Real evidence that it works.

The conventional wisdom: protect everything. Patent it. License it. Build walls. Extract value.

We chose different.

Why Open

Governance that hides defeats its purpose.

If an AI system claims to follow rules, you should be able to read those rules. If a validator claims compliance, you should be able to run that validator. If a specification claims completeness, you should be able to test that specification.

Trust requires transparency. Not as a slogan — as architecture.

What's Public

Everything that governs:

  • CANON.md — The constitutional specification
  • Validators — The enforcement tools
  • Documentation — How to implement
  • Examples — Reference implementations

The ledger. The evidence. The method.

What's Not

Some things remain private:

  • Patient data (obviously)
  • Active patent applications (until filed)
  • Commercial partnerships (until announced)

But the governance layer itself? Open. Because that's the point.

The Foundation Model

CANONIC Foundation is a 501(c)(3) non-profit. Not because we're anti-business — we have commercial tiers and enterprise partnerships. But because the core protocol belongs to everyone.

Like HTTP. Like TCP/IP. Like the protocols that built the internet.

Governance infrastructure should be a public good.

Join Us

The repositories are live:

  • canonic-foundation — Core specifications
  • canonic-apps — Reference applications

Issues are open. PRs are welcome. The conversation is public.

We're building governance for the AI era. We're doing it in the open. We hope you'll help.
